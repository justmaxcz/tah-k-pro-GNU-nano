Übersicht über die Tastenkombinationen von Nano      

Die Tastenanschläge des Editors und ihre Funktionen         


Dateiverwaltung
Ctrl+S   Aktuelle Datei speichern
Ctrl+OAngebot zum Schreiben der Datei ("Speichern unter")
Ctrl+RFügen Sie eine Datei in die aktuelle ein
Ctrl+XPuffer schließen, Nano verlassen

Bearbeitung
Ctrl+K   Aktuelle Zeile in Cutbuffer schneiden
Alt+6Aktuelle Zeile in Cutbuffer kopieren
Ctrl+UInhalt von Cutbuffer einfügen
Alt+TBis zum Ende des Puffers schneiden
Ctrl+]Vervollständigen Sie das aktuelle Wort
Alt+3Zeile/Bereich kommentieren/auskommentieren
Alt+ULetzte Aktion rückgängig machen
Alt+ELetzte rückgängig gemachte Aktion wiederherstellen

Suchen und ersetzen
Ctrl+Q   Rückwärtssuche starten
Ctrl+WVorwärtssuche starten
Alt+QNächstes Vorkommen rückwärts suchen
Alt+WNächstes Vorkommen weiter suchen
Alt+RStarten Sie eine Austauschsitzung

Streichung
Ctrl+HZeichen vor dem Cursor löschen      
Ctrl+DZeichen unter dem Cursor löschen
Alt+BspWort links löschen
Ctrl+Del   Wort rechts löschen
Alt+DelAktuelle Zeile löschen

Operations
Ctrl+T   Führen Sie einen Befehl aus
Ctrl+JBegründen Sie einen Absatz oder eine Region
Alt+JBegründen Sie den gesamten Puffer
Alt+BFühren Sie eine Syntaxprüfung durch
Alt+FFühren Sie einen Formatierer/Fixer/Arranger aus
Alt+:Aufzeichnung des Makros starten/stoppen
Alt+;Makro erneut abspielen

 Sich bewegen
Ctrl+B   Ein Zeichen rückwärts
Ctrl+FEin Charakter vorwärts
Ctrl+←Ein Wort rückwärts
Ctrl+→Ein Wort vorwärts
Ctrl+AZum Zeilenanfang
Ctrl+EBis zum Zeilenende
Ctrl+PEine Aufstellung
Ctrl+NEine Zeile tiefer
Ctrl+↑Zum vorherigen Block
Ctrl+↓Zum nächsten Block
Ctrl+YEine Seite nach oben
Ctrl+VEine Seite nach unten
Alt+\Bis zur Pufferoberkante
Alt+/Bis zum Ende des Puffers

Besondere Bewegung
Alt+G    Gehe zur angegebenen Zeile
Alt+]Gehen Sie zur Komplementärklammer
Alt+↑Scrollen Sie im Ansichtsfenster nach oben
Alt+↓Scrollen Sie im Ansichtsfenster nach unten
Alt+<Wechseln Sie zum vorherigen Puffer
Alt+>Wechseln Sie zum nachfolgenden Puffer

Informationen
Ctrl+C   Cursorposition melden
Alt+DZählen Sie die Anzahl der Zeilen/Wörter/Zeichen
Ctrl+GHilfetext anzeigen

verschieden
Alt+ASchalten Sie die Markierung ein/aus
TabMarkierten Bereich einrücken
Shift+Tab   Markierten Bereich rückgängig machen
Alt+VGeben Sie den nächsten Tastendruck wörtlich ein
Alt+NLeitungsnummern ein-/ausschalten
Alt+PSchalten Sie sichtbare Leerzeichen ein/aus
Alt+XBlenden Sie die Hilfslinien ein oder aus
Ctrl+LAktualisieren Sie den Bildschirm
